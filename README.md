# Penguins Dataset EDA

This project performs **Exploratory Data Analysis (EDA)** on the **Penguins dataset**, which contains measurements for penguins collected from the Palmer Station, Antarctica. The analysis includes **data cleaning, visualization, and insights**.

## Dataset

The dataset is available [here](https://raw.githubusercontent.com/mwaskom/seaborn-data/master/penguins.csv) and is also included as an **inbuilt dataset in Seaborn**.

### Columns

#### Categorical Columns
* `species` – Penguin species (Adelie, Chinstrap, Gentoo)  
* `island` – Island in the Palmer Archipelago  
* `sex` – Gender of the penguin  

#### Numerical Columns
* `bill_length_mm` – Bill length (mm)  
* `bill_depth_mm` – Bill depth (mm)  
* `flipper_length_mm` – Flipper length (mm)  
* `body_mass_g` – Body mass (g)
